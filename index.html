notes

📘 beginner sql
Basic SELECT query
SELECT customer_id, COUNT(*) as total_orders, SUM(amount) as total_spent FROM orders WHERE order_date >= '2025-01-01' GROUP BY customer_id ORDER BY total_spent DESC LIMIT 10;
Learn to aggregate and filter data. This query finds top customers by spending.
🐍 python basics
Pandas filter & groupby
import pandas as pd df = pd.read_csv('sales.csv') # Filter rows where revenue > 1000 filtered = df[df['revenue'] > 1000] # Group by region and calculate average result = filtered.groupby('region')['revenue'].mean().reset_index() print(result)
Essential pattern for data manipulation and aggregation in Python.
📊 window functions
Running total & ranking
SELECT date, product, sales, SUM(sales) OVER(PARTITION BY product ORDER BY date) as running_total, ROW_NUMBER() OVER(PARTITION BY product ORDER BY sales DESC) as rank_in_product FROM daily_sales ORDER BY product, date;
Window functions let you calculate running totals and rankings without subqueries.
🔄 etl pattern
Incremental load
def incremental_load(new_data, existing_table, key_column='id', timestamp_column='updated_at'): # Get latest timestamp from existing data latest = existing_table[timestamp_column].max() # Only get new or updated records new_records = new_data[new_data[timestamp_column] > latest] # Append to table return pd.concat([existing_table, new_records], ignore_index=True)
Efficient ETL pattern that only processes changed data, saving time and resources.

snippets

📅 date functions
SQL date truncation
-- Group by month, quarter, or year SELECT DATE_TRUNC('month', transaction_date) as month, SUM(amount) as monthly_revenue FROM transactions GROUP BY 1 ORDER BY 1 DESC;
Use DATE_TRUNC to aggregate time series data by month, week, or quarter.
🔗 join patterns
LEFT JOIN with COALESCE
SELECT u.user_id, u.name, COALESCE(o.total_orders, 0) as order_count, COALESCE(o.total_spent, 0) as lifetime_value FROM users u LEFT JOIN ( SELECT user_id, COUNT(*) as total_orders, SUM(amount) as total_spent FROM orders GROUP BY user_id ) o ON u.user_id = o.user_id;
LEFT JOIN keeps all users, COALESCE replaces NULL with 0 for cleaner results.
📈 python visualization
Quick matplotlib chart
import matplotlib.pyplot as plt import seaborn as sns # Simple bar chart df.groupby('category')['sales'].sum().plot(kind='bar') plt.title('Sales by Category') plt.xlabel('Category') plt.ylabel('Total Sales') plt.tight_layout() plt.show()
Quick exploratory visualization to spot trends and outliers in your data.
🧹 data cleaning
Handle missing values
# Python pandas - handle missing data # Fill numeric nulls with median df['price'].fillna(df['price'].median(), inplace=True) # Fill categorical with mode df['category'].fillna(df['category'].mode()[0], inplace=True) # Drop rows missing critical columns df.dropna(subset=['customer_id', 'transaction_date'], inplace=True)
Always clean missing values before analysis. Choose fill strategy based on data type.
